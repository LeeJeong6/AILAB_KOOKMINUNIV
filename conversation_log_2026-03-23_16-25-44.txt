[사용자 질문]
현재 폴더는 github page로 관리하는 연구실 스터디 폴더임. 학문적/논문 느낌으로 디자인을 유지하되 디자인을 다르게 하고싶음. 누구나 이 페이지에 pdf 또는 ppt를 upload할 수 있음. upload할 때 날짜를 선택(년-월-일)할 수 있고 발표자, 제목, 카테고리를 선택할 수 있음. 카테고리는 "논문 세미나", "바이브코딩" "수학스터디" 3개만 유지하기. 이때 누군가 파일을 업로드 하고 싶으면 해당 카테고리 내에 들어간 뒤 업로드 할 수 있는 형식이어야 함. 다음으로는 실제 연구실 "https://ailab.kookmin.ac.kr/" 으로 접속 할 수 있는 링크도 크게 있으면 좋겠고 현재 페이지에서 사진도 내가 직접 첨부해서 타이틀 화면에 보이게끔 하고싶음.

[어시스턴트 답변]
요청사항을 반영해 `index.html`을 전면 개편했습니다.
- 학문적 톤을 유지한 새 메인 디자인 적용
- 카테고리 3개(논문 세미나/바이브코딩/수학스터디) 고정
- 카테고리 입장 후 해당 섹션 내 업로드만 가능하도록 구현
- 업로드 필드: 날짜(YYYY-MM-DD), 발표자, 제목, 파일(PDF/PPT/PPTX)
- 업로드 항목은 IndexedDB에 저장되어 같은 브라우저에서 유지
- 타이틀 화면 이미지 직접 첨부/초기화 기능 추가
- 연구실 홈페이지 대형 링크(`https://ailab.kookmin.ac.kr/`) 추가
- 기존 샘플 페이지 뒤로가기 링크 수정

추가로 문서 작업도 완료했습니다.
- `README.md`: 새 계획 문서로 재작성
- `RESULT.md`: 결과 및 분석(한국어) 작성
